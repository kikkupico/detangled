#+title: detangled

* Prototype
the prototype is a front end application that can take as input a yaml pasted in a text area box and shows it as an explorable graph. The graph is navigated in a unique way.
** yaml format
detangled yaml represents an array of node objects with optional parent references. eg
#+begin_src yaml
meta:
  title: A Node graph
  description: |
    some desc
nodes:
 node1:
   title: Node 1
   short_title: node1
   emoji: "\uD83D\uDCA1"
   content: |
    node 1 content...
   parents: []
  node2:
    title: Node 2
    short_title: node2
    content: |
      node 2 content
    parents: [node1, node3]
  node 3:
    title: Node 3
    short_title: node3
    content: |
      node 3 content
#+end_src
